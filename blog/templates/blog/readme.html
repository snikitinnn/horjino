Версия 2.4

Обновление дизайна и навигации

На главной странице есть доступ к 4-м основным разделам: События, Сборники гимнов, Статистика, Темы гимнов. Для удобства на главной странице находится 3 основные ссылки для быстрого доступа к содержанию смешанного хора, мужского хора и группы прославления. Там же находится еще одна ссылка "Старая версия" - если новая версия сайта вызовет затруднения, т.е. на всякий пожарный случай.

Основной принцип навигации

Из главной страницы можно попасть в любую часть базы данных связанной с конкретным служением согласно выбранному разделу.После этого в верхней части всегда будет доступна кнопка "Меню", c помощью которой можно будет перемещаться в нужный раздел данного служения. Кроме того, в некоторых разделах будут доступны общие для всех служений пункты меню, такие как "Сортировка по базе", "Полный список богослужений".
